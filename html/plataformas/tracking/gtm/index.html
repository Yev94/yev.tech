GTM desde el lado del cliente

El código GTM se instala en el cliente y envía los datos a las diversas plataformas

GTM desde el lado del servidor

El código GTM se encuentra tanto en el cliente como en el servidor, y el GTM en el cliente envía los datos al servidor.

Posteriormente, es el GTM del servidor el encargado de enviar los datos a las distintas plataformas.

El envío de datos puede ser del cliente al GTM servidor directamente ó del cliente a nuestro servidor y posteriormente al GTM servidor

Con este método podemos saltarnos a los bloqueadores de anuncios (de momento).
