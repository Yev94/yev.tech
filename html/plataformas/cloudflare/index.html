Pasar por consola la siguiente comprobación

curl -X PATCH "https://api.cloudflare.com/client/v4/zones/< Zone ID >" -H "X-Auth-Email: sandra@muemue.com" -H "X-Auth-Key: < GLOBAL API KEY HERE >" -H "Content-Type: application/json" -d '{"status":"pending"}'

Donde < GLOBAL API KEY HERE > está en: Usuario > Appearance > Menú(API Tokens) > Global PI Key > View

Donde < Zone ID > está en: el dominio > Menú(Overview) > En la parte derecha API >Zone ID
