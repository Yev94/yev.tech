PHP My Admin

Instalar PHP My Admin en WAMP

  1. Descargar
  2. Extraer archivos en: C:\wamp64\apps
  3. Crear archivo alias siguiendo la estructura del resto en: C:\wamp64\alias

    4. Es Para crear la URL, por lo que se puede cambiar la anterior para que la nueva sea /phpmyadmin

  4. Habilitar inicio sesión sin contraseña yendo a: C:\wamp64\apps\phpmyadmin5.2.0
    1. Crear config.inic a partir de config.sample.inic
    2. Buscar: $cfg['Servers'][$i]['AllowNoPassword']
    3. Cambiar a true
