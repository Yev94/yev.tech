Librerías Interesantes

PlayWright

Playwright es una librería de Node.js que permite automatizar navegadores web. Playwright está diseñado para admitir múltiples navegadores (Chromium, Firefox y WebKit) y ejecutar páginas web en paralelo.

Leaflet

Leaflet es la principal biblioteca de código abierto para mapas interactivos. Lo que comenzó como una biblioteca para la plataforma de software de mapas en línea CloudMade en 2010 se ha convertido en una de las bibliotecas de código abierto más populares para mapas interactivos.

Ref: Mapas con tan SOLO UNA LÍNEA de JavaScript y más 🤯

GSAP

Es una biblioteca de JavaScript de código abierto para crear animaciones de alta calidad y alto rendimiento en navegadores modernos. GSAP es una biblioteca de animación de JavaScript de alto rendimiento que funciona con todos los principales navegadores.

Three.js

Es una biblioteca de JavaScript de código abierto utilizada para crear y mostrar gráficos animados en 3D en un navegador web. Three.js utiliza WebGL.

Chart.js

Es una biblioteca de gráficos de JavaScript gratuita y de código abierto que se puede utilizar para crear gráficos dinámicos y hermosos para representar datos.

Socket.io

Es una biblioteca de JavaScript para aplicaciones web en tiempo real. Permite la comunicación bidireccional en tiempo real entre clientes y servidores web. Tiene dos partes: una biblioteca del lado del cliente que se ejecuta en el navegador y una biblioteca del lado del servidor para Node.js.
