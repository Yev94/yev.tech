Conexiones Archivos Import/Export

Importante que import arriba del archivo y en el script del HTML tenga atributo type='module'

Importar Todo

        
importScripts(s1.js, s2.js, ...);
import * as alias from "./ruta";
alias.funcion();

//Para Extensiones chrome
(async () => {
      const src = chrome.runtime.getURL("your/content_main.js");
      const contentMain = await import(src);
      contentMain.main();
})();

Importar Función en Específico

        
import {fun, ...} /as alias/ from "./ruta";//Alias opcional

Exportar Funciones

        
export funcion();
export {funcion, funcion2, funcion3...}
