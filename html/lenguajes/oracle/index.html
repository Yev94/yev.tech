Según Campos
Seleccionar campos tabla SELECT camp FROM nomTab
Todos los campos camp = *
Varios campos concretos camp = nomCamp1, nomCamp2...
Ver campo con fórmula camp = nomCamp*2 AS campDoble
Ver campo sin repetir val camp = DISTINCT nomCamp
Según Valor (Where)
Seleccionar campos tabla SELECT camp FROM nomTab WHERE condiW
Que val campo > x condiW = nomCamp > x
Que val camp esté entre x,y condiW = nomCamp BETWEEN x AND y
Que val camp NO entre x,y condiW = nomCamp NOT BETWEEN x AND y
Que val camp sea x ó y condiW: nomCamp = x OR nomCamp = y
Que val camp sea x ó y ó z ó... condiW = nomCamp IN (x, y, z ...)
Algún campo termine en z condiW = nomCamp1 LIKE '%z' OR nomCamp2 LIKE '%z';
Cualquier char anterior-post CharComodin = %
Cualquier char del indice CharComodin = _
Que el valor sea null/No null condiW = nomCamp IS NULL / NOT NULL
