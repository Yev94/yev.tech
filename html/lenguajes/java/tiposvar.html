Tipos de Variable

Funciones

Función (void) que NO devuelve valor public void miFuncion(){}
Función que Sí devuelve valor public int miFuncion(int a){return a} // Si declaro que fun es int debe devolver int
Función (estática) que se puede llamar desde clase u objeto public static miFuncion(){}; // puede ser llamada desde:
// miClase.miFuncion();
// miOBjeto.miFuncion();

General

Objeto padre de todos que permite apuntar a cualquier objeto Object bar = new MiObjeto1(10)
// Luego podríamos hacer: bar = new MiObjeto2();
// De normal sería: MiObjeto1 bar = new MiObjeto1(10);
// Luego siempre: bar = new MiObjeto1(2);
Constante final double bar = 8.123456789012345
Estáticos static int bar = 2,147,483,647
Booleano escalar (no objeto)(tipo primitivo) boolean bar = true

Numéricos (escalares (no objetos)(tipos primitivos))

Entero (4 bytes) int bar = 2,147,483,647
Corto (2 bytes) short bar = 32,767
Largo (8 bytes) long bar = 2,147,483,648// se puede poner una l al final
Byte (1 byte = 8 bits) byte bar = -127
Flotante (4 byte (aprox 6 decimales)) float bar = 8.123456f
Flotante Extendida (8 byte (aprox 15 decimales)) double bar = 8.123456789012345 // se puede poner un d al final

Ref: Curso Java. Estructuras principales II. Vídeo 5

Caracteres

Carácter (tipo primitivo) char bar = 'a' // Siempre entre comillas simples
String (Clase Objeto Predefinido) String bar = "a"
