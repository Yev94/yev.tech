Enumerate

Un objeto enumerate es un iterador que devuelve una secuencia de pares (índice, valor) a partir de otro iterable

Para acceder al mismo se puede convertir en una lista o una tupla con las funciones list() o tuple()

Pero es más eficaz recorrerlo con un for

Librerías

  • Math: Librería primitivas matemáticas nativas de Python.
  • Sympy: Librería para facilitar la realización de cálculos matemáticos con variables sin necesidad de definir funciones nativas de Python.
    • from sympy.plotting import plot as symplot # Librería para las gráficas
    • from sympy.abc import x, y, h # Carga de un simbólico "x"
    • from sympy.plotting.pygletplot import PygletPlot as Plot # Librería para las gráficas
  • Scipy: Librería con diversas funciones para matemáticas e ingeniería.
  • Numpy: Librería para realizar cálculos y manipulación numérica.
  • Matplotlib: Librería para gráficos.
