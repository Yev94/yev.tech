Multiciclo

Permite tener distintos ciclos para cada instrucción ya que se ejecuta cada uno en distintos ciclos

Permite unificar hardware que en la monociclo tocó duplicar

En el control de cada ciclo puedo activar elementos que podrían ir en otro ciclo pero que podría usar en uno anterior para ahorrar ciclos

Se añaden Memorias intermedias para usarlas en el ciclo
