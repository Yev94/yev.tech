Frases Trikies

  • Es cierto A a menos que sea falso B

    • ¬A → ¬B

    Viene de: No A a menos que B (A → B)

  • A menos que sea cierto B, es falso A, pero cierto C

    • ¬(¬A ∧ C) → B

    Viene de: No A a menos que B (A → B)

  • Si es cierto A y B, no lo es C ni D

    • A ∧ B → ¬C ∧ ¬D

    Viene de: si A entonces B

  • Sólo si es falso A o falso B, pero cierto D, entonces es cierto B

    • B → ( ¬A ∨ ¬B ) ∧ D

    Viene de: A solo si B

  • No es suficiente, pero sí necesario, que sea cierto A o B para que sea cierto C

    • ¬ ( A ∨ B → C ) ∧ ( C → A ∨ B )

    Viene de:

    • A es suficiente para B (→)
    • B es necesario para A (→)
    • A pero B (∧)
  • No es necesario que sea falso B, pero cierto C, para que sea cierto A

    • ¬( A → ¬B ∧ C )

    Viene de: B es necesario para A

  • Si es cierto A y B entonces, si no es cierto C, pero sí lo es D tenemos que es falso A o es falso B

    • A ∧ B → (¬C ∧ D → ¬A ∨ ¬B)
