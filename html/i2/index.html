La idea es que, en vez de tener que estar todo el tiempo repasando para que quede claro (como a l/p no se sabe si va a ser útil o no el conocimiento adquirido), se trata de tener una forma de volver a tener la misma fluidez que cuando se estaba estudiando.

Para ello se crean, a través de las distintas herramientas disponibles, una forma de volver a adquirir (del modo más rápido posible) el conocimiento, adaptado a la forma de entender de cada uno.

A partir de aquí se trata de ir mejorando la eficiencia de cada proceso, adaptándolo a las distintas necesidades
