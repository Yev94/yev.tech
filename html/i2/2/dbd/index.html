Diseño de Bases de Datos

Conceptos

Base de Datos Conjunto de datos relacionados entre sí
DBMS Sistema de Gestión de Bases de Datos
Modelo de Datos Conjunto de conceptos que permiten describir la estructura de una base de datos
Modelo de Datos Conceptual Modelo de datos que describe la estructura de una base de datos sin hacer referencia a un SGBD concreto
Modelo de Datos Lógico Modelo de datos que describe la estructura de una base de datos haciendo referencia a un SGBD concreto
Modelo de Datos Físico Modelo de datos que describe la estructura de una base de datos haciendo referencia a un SGBD concreto y a un sistema operativo concreto
Modelo de Datos Jerárquico Modelo de datos en el que los datos se organizan en forma de árbol
Modelo de Datos en Red Modelo de datos en el que los datos se organizan en forma de grafo
Modelo de Datos Relacional Modelo de datos en el que los datos se organizan en forma de tablas
Modelo de Datos Orientado a Objetos Modelo de datos en el que los datos se organizan en forma de objetos
