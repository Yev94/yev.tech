Gestión de Procesos

Ref: Gestión de procesos (ISO - 3.2)

Bloque de Control de Procesos

Estructura de datos del sistema que mantiene toda la información sobre un proceso.

Se crea cuando se crea un proceso y se destruye cuando el proceso termina.

Info que contiene:

  • Identificador del proceso: PID, PPID (Nª del proceso padre)…
  • Estado de un proceso:
    • Espera
    • Preparado
    • Ejecución
    • Bloqueado ó en Espera (La impresora se queda sin papel y toca es esperar a imprimir)
    • Espera
  • Registros de la CPU: CP, flags
  • Info de planificación de la CPU: prioridad, …
  • Info de gestión de memoria: punteros, tablas, registros, …
  • Información contable: tiempo consumido, límites de tiempo, …
  • Información de estado de E/S: lista de dispositivos de E/S, lista de archivos abiertos

    • Ref: Bloques de Control de Procesos (BCP)

    Ref: Gestión de memoria RAM (ISO - 3.3)
