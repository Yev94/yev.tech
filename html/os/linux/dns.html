DNS

Comandos básicos para ver info

Estos comandos permiten ver información que tiene el DNS sobre domino en concreto

Esta información viene dada por las siguientes siglas

  • A - IPv4
  • AAAA - IPv6
  • MX - Mail Exchange (el dominio del mail)
  • NS - Name Server (El servidor autorizado para resolver el nombre del dominio de la IP)
  • CNAME(Canocical Name - Alias) - Permiten redirigir un subdominio a otro, por ejemplo, ftp.heey.com a www.heey.com
  • PTR - Asocia una IP asociada al nombre del DNS (equivalente a cuando hacemos la petición en nslookup de nombre de dominio a través de su IP), por lo que también es conocido como registro inverso

Para obtener esta información tenemos que usar los comandos del CMD

Windows: nslookup nomServidor/IPServidor nomDNS/IPDNS(Opcional, sino coge el que tenemos configurado por defecto)

Cuando hagamos la petición la respuesta será de un servidor DNS no autorizado

Para encontrar el autorizado solo hay que ver los servidores que aparecen la primera vez al final de información arrojada por la petición

Luego solo tenemos que introducirla como nomDNS/IPDNS

Linux: dig @nomDNS/IPDNS any/A/MX... nomServidor
