Generales

  • Prettier

    Formatea el código para que quede más legible

    "alt+shift+f"

  • vscode-icons

    Pone iconos para los distintos tipos de archivos

  • open in browser

    Abre el código en el navegador

  • Live Server

    Abre el código en el navegador en forma de servidor, actualizando todos los cambios del código automáticamente

    También sirve para debuggear JS lanzándolo al navegador

  • Tabnine AI Autocomplete y Kite AutoComplete AI Code

    IntelliSene con IA

  • Code Spell Checker (+Esp)

    Checkea los errores sintácticos en inglés y español

  • Better comments

    Permite que se cambie el color de los comentarios en función del tipo de comentario que se le asigna al principio

    * para info importante

    ! para función obsoleta, no usar

    ? para preguntarse si el factor debería ser visto por el público

    TODO: para funciones a refactorizar

    @param para explicar el parámetro para el método

  • Rainbow Brackets

    Colorea cada par de signos de grupo como paréntesis, corchetes, etc, permitiendo ver qué agrupan los signos

  • Indent-rainbow

    Colorea las indentaciones

  • Peacock

    Cambia el color de los bordes de la ventana para tener más claro en qué ventana se encuentra cada proyecto

  • Settings Sync

    Crea un archivo de sincronización en Github Gist

    "shift+alt+u" para subir

    "shift+alt+d" para bajar

  • Path IntelliSense

    IntelliSense para ver lo que hay en el directorio sin que tenga que detectar el slash

  • Error Lens

    Enseña al final de la línea el tipo de error que te muestran los Lens de cada Lenguaje

  • Run on save by pucelle

    Ejecuta en la terminal cualquier comando al guardar

    Requiere insertar la configuración deseada en el settings.json

  • Editor config for VS CODE

    Permite, a través un archivo de configuración creado con la extensión de configuración, tener una misma configuración en cualquier nuevo editor

  • Bookmarks

    Crea un bookmark de linea para ir resaltando y saltando entre ellas

  • TODO Tree

    Permite colorear los TODO y los FIXME

    También permite tenerlos listados en la barra lateral izq

  • Code Runner

    Permite ejecutar el código en la terminal, desde todo el código hasta solo un trozo seleccionado

    Con ctrl+alt+k ejecuta un comando personalizado en la terminal

    Pero como la extensión de bookmarks tiene el mismo shortcut lo cambio a ctrl+alt+i

    Tambien se puede ejecutar con F1 > 🔎Run Custom Command

Formatos y herramientas para código específico

  • Python y Pylance
  • SASS
  • PHP IntelliSense, PHP Intelephense

JS

  • Quokka y quokka status bar icons

    Ejecuta el código internamente mientras se va creando para ver qué salidas dá

    La segunda pone botones abajo que ayudan a ejecutar la extensión

  • Babel

    Resalta código JS ES6

  • ESLint

    Corre por detrás tratando de reconocer errores sintácticos en el código

PHP

  • PHP Server

    Crea un servidor virtual para ejecutar el PHP al estilo XAMPP

  • PHP Debug Server

    Requiere configuración

    1. Ir a XDebug
    2. Copiar archivo PHP info (si usas XAMPP este se encuentra en dashboard > phpInfo)
    3. Pegarlo en el Wizard de XDebug
    4. Descargar el dll y seguir las instrucciones
    5. Añadir las lineas para configurar el debug remoto (instrucciones en la descripción de la extensión)
    6. Configurar el archivo de debugger para XDebug

HTML

  • Auto Rename Tag

    Cambia una la etiqueta correspondiente al cambiar la etiqueta de apertura o cierre

  • html tag wrapper

    Crea div agrupando la selección

    Se puede cambiar la etiqueta

    "ctrl+i"

SQL

  • MySQL

    Permite conectarte a la base de datos desde VS como si fuera PHPMyAdmin o Workbench

CSS

  • Intellisense for class names in HTML

    Sugiere nombres de clases (Bootstrap)

    Con "ctrl+space"

  • Color highlight

    Colorea el num hexadecimal del color que pongas el num

  • css-auto-prefix

    Añade las declaraciones en aquellas en las que en los antiguos navegadores tenían conflictos

  • Intellisense for CSS

Snippets

  • HTML Snippets

  • Git Lens

    Permite ver en forma de comentario en la línea seleccionada quién ha hecho los cambios de esa línea
